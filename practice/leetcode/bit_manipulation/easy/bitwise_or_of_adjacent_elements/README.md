# 3173. Bitwise OR of Adjacent Elements

**[Solution](./solution.c) - [Notes](./notes.md)**

## Description
Given an array `nums`, return a new array where each element is the bitwise OR of adjacent elements in `nums`.

## Examples

### Example 1
**Input:** `nums = [1,3,7,15]`  
**Output:** `[3,7,15]`

### Example 2
**Input:** `nums = [8,4,2]`  
**Output:** `[12,6]`

### Example 3
**Input:** `nums = [5,4,9,11]`  
**Output:** `[5,13,11]`

## Constraints
- `2 <= nums.length <= 100`
- `0 <= nums[i] <= 100`
