# 266. Palindrome Permutation

**[Solution](./solution.c) - [Notes](./notes.md)**

## Description
Given a string `s`, determine if a permutation of the string could form a palindrome.

## Examples

### Example 1
**Input:** `"code"`  
**Output:** `false`

### Example 2
**Input:** `"aab"`  
**Output:** `true`

### Example 3
**Input:** `"carerac"`  
**Output:** `true`

## Constraints
- `1 <= s.length <= 5000`
- `s` consists of lowercase English letters
